Instalar extensões Thunder Client, Prisma e Prisma Insider
Iniciar projeto: npm init -y
Instalar express: npm i express
Instalar dotenv: npm i dotenv

Adicionar no package.json:
"type":"module"
"dev": "node --watch index.js"
"start": "node index.js"

Criar um arquivo index.js

MongoDB > Projects > New Project > Create a Project > Create a cluster > Free > Editar Cluster name > Create deployment > Anotar Username e Password  > Create Database User > Choose a connection method > Drivers > Node.js > Anotar URL.

Prisma.io > Get started > Start from scratch > MongoDB > Segue o passo a passo da documentação.
